# 📂 Smart File Organizer（智慧型檔案自動分類器）

一個使用 **Python** 開發的輕量化自動檔案整理工具。

只需要將 `main.py` 放到想整理的資料夾（例如 Downloads），執行一次程式，即可依照檔案副檔名自動建立分類資料夾，讓凌亂的資料夾瞬間變得井然有序。

---

## ✨ 功能特色

*  **自動偵測目前資料夾**

  * 不需要修改任何路徑設定。
  * 程式會自動整理 `main.py` 所在的資料夾。

* **安全防護機制**

  * 不會移動或刪除 `main.py` 自己。
  * 自動略過所有資料夾，只整理一般檔案。

* **智慧副檔名辨識**
  * 自動衝突解決：透過自動遞增重複的檔名來防止檔案被覆蓋。
  * 重複檔案處理：如果已有同名檔案存在，會自動加上字尾重新命名(指 _1, _2 這種作法)。
  * 自動將副檔名轉成小寫比對。
  * `.JPG`、`.jpg`、`.Jpg` 都會被視為相同格式。

* **自動建立分類資料夾**

  * 若分類資料夾不存在，程式會自動建立。

## 📁 執行效果

### 執行前（Before）

```text
your_folder/
├── main.py
├── wallpaper.jpeg
├── invoice.pdf
├── backup.zip
└── report.docx
```

### 執行後（After）

```text
your_folder/
├── main.py
├── 圖片/
│   └── wallpaper.jpeg
├── 文件/
│   ├── invoice.pdf
│   └── report.docx
└── 壓縮包/
    └── backup.zip
```

## 🚀 支援的檔案類型

| 分類            | 支援副檔名                        |
| ------------- | ---------------------------- |
| 🖼️ 圖片        | `.jpg` `.jpeg` `.png` `.gif` |
| 📄 文件         | `.txt` `.pdf` `.docx`        |
| 📊 簡報         | `.pptx`                      |
| 🎵 音樂 / 🎬 影片 | `.mp3` `.mp4`                |
| 📦 壓縮包        | `.zip`                       |
| 💿 安裝檔        | `.exe`                       |

> 未包含於以上列表的檔案會保留在原位置，不會被移動。

## 🛠️ 使用方式

### 1. 將程式放入目標資料夾

例如：

```text
Downloads/
├── main.py
├── photo.jpg
├── file.pdf
└── music.mp3
```

### 2. 開啟 Terminal

切換到該資料夾：

```bash
cd Downloads
```

### 3. 執行程式

```bash
python3 main.py
```

### 4. 完成！

若看到終端機輸出：

```text
🎉 Stop scanning, organization completed!
```

代表檔案已成功完成分類。
